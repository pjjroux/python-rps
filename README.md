# python-rps

Rock, paper, scissors CLI game in Python

## Description

We are building the project as part of learning Python 3.

## Getting Started

### Dependencies

* [Python 3](https://www.python.org/)

### Installing

* Clone or download the project files to your local machine. 

### Executing program

* Run main.py
```
* Commands
> R or ROCK
> P or PAPER
> S or SCISSOR
> E or EXIT
```
